环境系统

Air Quality System

环境系统通过与探测器的联动，进行实时监测，并借助相关设备自动调节空气质量。包括监测甲醛、PM2.5、湿度等在内的环境数据，并能够与场景组合联动。

灯控系统

Lighting Control System

灯控系统通过数字智能网关、智能开关、智能家居遥控器、智能灯光遥控器、智能灯光控制器的结合，实现对灯光的智能管理。包括：控制灯光的开和关，调节灯光的亮度，灯光颜色的变换、实现各种灯光情景的变换。

安防系统

Security System

安防系统以运用安全防范产品和其它相关产品所构成的入侵报警系统、视频安防监控系统、出入口控制系统、防爆安全检查等的系统。完整的智能安防系统包括门禁、报警和监控。

中控系统

Central Control System

中控系统指对声、光、电等各种设备进行集中控制的设备。包括灯光、窗帘、空调、地暖、投影机、影碟机、录像机、视频展示台、多媒体电脑、电动屏幕等。

窗帘系统

Curtain Control System

窗帘系统是指利用窗帘控制器、光线感应技术以及无线传输技术，对窗帘实现自动调节及控制。

温控系统

Temperature Control Units

温控系统是一种具有室内温度调节功能的智能家居控制系统，包括：空调控制器、感应装置和温度控制面板等。

影音系统

Audio-Visual System

影音系统能够统一管理与控制家里的各种影音设备。包括：功放，音响，高清播放机，投影机，投影幕，高清电视等。

ZigBee

ZigBee是基于IEEE802.15.4标准的低功耗局域网协议，是一种短距离、低功耗的无线通信技术。适合用于自动控制和远程控制领域，可以嵌入各种设备。
